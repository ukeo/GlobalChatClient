# GlobalChat Client | Global Chat solution for spigot

This is the documentation about GlobalChat.

## Getting Started

Download the plugin [HERE](https://spigotmc.org) and put it on your server.

Start chatting with !gc [Message]

### Prerequisites

Spigot 1.8-1.19
